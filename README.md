# ❓ Quiz de Perguntas

Teste seus conhecimentos num quiz de múltipla escolha, todo em português. Seis categorias, feedback na hora e seu recorde guardado — offline e sem cadastro.

## O que faz

- **6 categorias**: Conhecimentos Gerais, Ciência, História, Geografia, Esportes e Entretenimento
- Opção de **misturar todas** as categorias numa partida
- **Feedback imediato** — a alternativa certa e a errada se acendem na hora
- **Pontuação** e **barra de progresso** ao longo da partida
- **Recorde** guardado no navegador
- Perguntas e alternativas **embaralhadas** a cada partida
- 100% **responsivo** e **offline**

## Sobre as perguntas

O banco de perguntas é **curado em português** — nada de tradução automática. As alternativas são embaralhadas a cada rodada, então a mesma pergunta nunca vem igual.

## Stack

Next.js 16 (App Router) · TypeScript · Tailwind CSS v4 · Framer Motion · Lucide. Sem banco e sem API — as perguntas vivem no próprio projeto; o recorde fica no `localStorage`.

## Rodar localmente

```bash
npm install
npm run dev
```

Abre em `http://localhost:3000`.
